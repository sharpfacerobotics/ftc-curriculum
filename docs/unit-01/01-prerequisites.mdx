# Section 1: What to Have Ready Before You Start

:::info Read before you begin
This manual assumes you have never installed Android Studio before. Every step is numbered. Every common failure has a labeled fix. Read each step completely before executing it. Do not skip steps.
:::

---

## Hardware Required

- A laptop or desktop running Windows 10/11, macOS 12+, or Ubuntu 20.04+
- Minimum 8 GB RAM (16 GB strongly recommended)
- Minimum 10 GB free disk space
- An internet connection for the initial setup

## What You Will Install

- Android Studio (Ladybug 2024.2.1 or newer)
- JDK 17 (manually configured — do not use Android Studio's bundled JDK)
- The FTC SDK (cloned from GitHub)
